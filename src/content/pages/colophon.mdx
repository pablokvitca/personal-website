This website is built with modern web technologies, prioritizing performance, accessibility, and developer experience.

## Technology Stack

### Framework

- **[Astro](https://astro.build)** - Static site generator with islands architecture
- **[React](https://react.dev)** - For interactive components
- **[TypeScript](https://www.typescriptlang.org)** - Type-safe development

### Styling

- **[Tailwind CSS](https://tailwindcss.com)** - Utility-first CSS framework
- **[Starwind](https://starwind.dev)** - Astro component library

### Content

- **[MDX](https://mdxjs.com)** - Markdown with JSX components
- **Astro Content Collections** - Type-safe content management

## Hosting & Infrastructure

- **[Cloudflare Pages](https://pages.cloudflare.com)** - Edge deployment and hosting
- **[Cloudflare Web Analytics](https://www.cloudflare.com/web-analytics/)** - Privacy-first analytics

## Security

- **[Arcjet](https://arcjet.com)** - Application security

## Analytics

- **[PostHog](https://posthog.com)** - Product analytics and event tracking
- **[Cloudflare Analytics](https://www.cloudflare.com/web-analytics/)** - Basic web analytics

## Development

- **[pnpm](https://pnpm.io)** - Package manager
- **[mise](https://mise.jdx.dev)** - Runtime version management
- **[Git](https://git-scm.com)** - Version control with conventional commits
